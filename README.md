# 꼬꼬 사용 설명서

이 설명서를 따라하면 꼬꼬 사이트를 쉽게 사용할 수 있어!

---

## 메인 페이지 (방문자용)

메인 페이지에서는 선택한 귀여운 이미지를 볼 수 있어!

### 좋아요 누르기

1. 화면에 보이는 이미지를 **빠르게 두 번 톡톡** 터치해!
2. 하트가 뿅! 하고 나타나면 좋아요 성공!
3. 아래에 있는 하트 숫자가 올라가는 걸 확인해봐

> 참고: 좋아요는 10분에 한 번만 누를 수 있어. 너무 빨리 다시 누르면 "잠시만 기다려주세요!" 라는 메시지가 나와!

---

## 관리자 페이지

### 로그인하기

1. 오른쪽 구석 관리자 글자를 클릭하거나 주소창 뒤에 `/login`을 붙이거나서 들어가 (예: `사이트주소/login`)
2. 이메일과 비밀번호를 입력해
3. "로그인" 버튼을 눌러!

---

### 이미지 관리 탭

여기서 이미지를 올리고 관리할 수 있어!

#### 이미지 올리기
1. "파일 선택" 버튼을 눌러
2. 컴퓨터에서 올리고 싶은 사진을 골라
3. 사진이 자동으로 올라가!

#### 메인에 보여줄 이미지 선택하기
1. 이미지 목록에서 보여주고 싶은 사진을 찾아
2. 그 사진 아래에 있는 **"선택"** 버튼을 눌러
3. 이제 메인 페이지에서 그 사진이 보여!

#### 이미지 삭제하기
1. 삭제하고 싶은 사진을 찾아
2. **"삭제"** 버튼을 눌러
3. 사진이 사라져!

---

### 통계 탭

여기서 좋아요가 얼마나 눌렸는지 볼 수 있어!

- **날짜별 그래프**: 매일 좋아요가 몇 개 눌렸는지 볼 수 있어
- **이미지별 그래프**: 어떤 사진이 좋아요를 많이 받았는지 볼 수 있어
- 기간을 7일, 30일, 90일로 바꿔서 볼 수 있어!

---

### AI 생성 탭

AI가 귀여운 병아리 그림을 만들어줘!

1. 어떤 그림을 원하는지 설명을 적어 (예: "모자 쓴 병아리", "잠자는 병아리")
2. **"생성"** 버튼을 눌러
3. 조금 기다리면 AI가 그림을 만들어줘!
4. 마음에 들면 **"저장"** 버튼을 눌러서 이미지 목록에 추가해

---

### 로그아웃하기

관리자 페이지 오른쪽 위에 있는 **"로그아웃"** 버튼을 누르면 돼!

## 정리

| 하고 싶은 것 | 어떻게 하면 돼? |
|-------------|----------------|
| 좋아요 누르기 | 이미지 두 번 톡톡! |
| 관리자 들어가기 | 주소에 `/login` 붙이기 |
| 메인 이미지 바꾸기 | 이미지 관리 > 선택 버튼 |
| AI 그림 만들기 | AI 생성 탭 > 설명 쓰고 생성! |
| 통계 보기 | 통계 탭 클릭 |
